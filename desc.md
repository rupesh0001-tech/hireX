We are building a real-time meeting intelligence system that works seamlessly alongside platforms like Zoom, Google Meet, and Microsoft Teams — without requiring any change to existing workflows.

The system runs as a background companion, capturing live meeting audio to generate real-time transcriptions, while simultaneously identifying action items, decisions, and key discussion points as they happen. It intelligently assigns tasks to participants based on conversational context and mentions.

In parallel, it monitors the screen to capture timestamped screenshots whenever important visuals such as slides, diagrams, or shared documents appear — preserving context beyond text.

A live sidebar interface provides participants with a continuously updating summary and dynamic to-do list, ensuring everyone stays aligned throughout the meeting.

Once the meeting ends, the system automatically delivers a structured debrief to each participant, including:

Key decisions made
Open questions
A screenshot gallery
Personalized action items

It also detects follow-up discussions and creates calendar events automatically, while syncing tasks directly with tools like Notion, Jira, and Linear
